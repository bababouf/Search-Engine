Search Genie

Github Repo

This web application enables efficient search and indexing, offering quick access to large directories through a simple interface.

Corpus Selection

Users have the flexibility to choose between the default corpus or upload their own, catering to diverse research needs and interests.

Indexing Efficiency

Upon selecting a corpus, our application initiates a comprehensive indexing process, optimizing data for rapid retrieval and ensuring minimal latency during subsequent queries.

Query Functionality

The application supports two distinct query modes to accommodate varying search preferences:
1. Boolean Retrieval Mode
2. Ranked Retrieval Mode
